# Optima Staff Analytics

Готовый дипломный макет сайта для мониторинга рабочего времени и аналитики сотрудников банка.

## Что умеет проект

- панель управления с основными KPI
- учёт сотрудников
- учёт посещаемости и рабочего времени
- аналитика по дням, сотрудникам и филиалам
- ежемесячный отчёт
- экспорт CSV
- резервное копирование JSON
- хранение данных в LocalStorage
- дизайн в банковом стиле, похожий на современную панель Optima

## Технологии

- HTML
- CSS
- JavaScript
- LocalStorage
- SQL-файл для будущей серверной версии

## Как открыть

1. Распакуй проект
2. Открой папку в Visual Studio Code
3. Запусти `index.html`
4. Лучше использовать расширение **Live Server**

## Как загрузить на GitHub

1. Создай новый репозиторий
2. Загрузи файлы `index.html`, `styles.css`, `app.js`, `README.md`, `database.sql`
3. Включи GitHub Pages в настройках репозитория
4. Выбери ветку `main` и папку `/root`
5. После публикации сайт будет открываться по ссылке GitHub Pages

## Важно

Текущая версия полностью работает как фронтенд-приложение в браузере.
Для реального банковского внедрения потом можно добавить:

- авторизацию
- роли пользователей
- серверную часть
- API
- настоящую SQL базу данных
- журнал действий пользователей
- интеграцию с турникетами или внутренней системой банка

## Рекомендуемое развитие для диплома

- сделать страницу входа
- добавить графики через Chart.js
- сделать backend на Node.js + Express
- подключить MySQL или PostgreSQL
- добавить PDF-отчёты
- разделить систему на роли: администратор, HR, менеджер
